==============================
How to Use MyList Class
==============================

This module defines an inherited class called MyList.

MyList Class
=============

The `MyList` class is an inherited class from the built-in `list` class. The method `print_sorted()` is added to print the list in ascending sorted order.

Usage
=====

To use the `MyList` class, follow these steps:

1. Simply create an instance of the `MyList` class

my_list = MyList()
